# ربات فروش VPN - راهنمای دیپلوی

## روش استقرار رایگان روی Railway

### مرحله ۱ — آماده‌سازی GitHub
1. یه repo جدید توی GitHub بساز (خصوصی)
2. این ۳ فایل رو آپلود کن:
   - `bot.py`
   - `requirements.txt`
   - `Procfile`

### مرحله ۲ — دیپلوی روی Railway
1. برو به [railway.app](https://railway.app)
2. با GitHub لاگین کن
3. **New Project → Deploy from GitHub repo** رو بزن
4. repo ربات رو انتخاب کن
5. بعد از ساخت پروژه، برو تب **Variables** و این متغیرها رو اضافه کن:

```
BOT_TOKEN=8537532580:AAEHXHa9a_oS5-iPYdx-KEk6boD4Mea5UH4
ADMIN_ID=474098524
```

6. توی تب **Settings → Service** مطمئن شو که **Start Command** اینه:
   ```
   python bot.py
   ```

### مرحله ۳ — تست ربات
- ربات رو توی تلگرام پیدا کن: @Turk_nettbot
- /start بزن

## جریان کاری ربات

```
کاربر: /start
  ↓
منوی اصلی (خرید / پکیج‌ها / پشتیبانی)
  ↓
انتخاب پکیج
  ↓
نمایش اطلاعات کارت برای واریز
  ↓
کاربر عکس رسید می‌فرسته
  ↓
ادمین (474098524) نوتیف می‌گیره + دکمه تایید/رد
  ↓
ادمین تایید → کانفیگ رو می‌فرسته به ربات
  ↓
ربات کانفیگ رو به کاربر می‌فرسته
```
